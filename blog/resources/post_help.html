Post Parameters

These parameters are available for all Post types:

blog, state, tags, date

Get more information about parameters on official tumblr documentation, you can find it here.

Text Posts

Parameters implemented: title, body

resource "tumblr_post_text" "first_text" {
    blog  = "my_blog"
    title = "First text applied by terraform-provider-tumblr"
    body  = "First text applied by terraform-provider-tumblr."
    state = "published"
    tags  = "terraform,terraform provider,tumblr,text"
    date  = "2006-01-02 15:04:05 GMT"
}
